Given an integer `n`, return *he number of structurally unique **BST**'s (binary search trees) which has exactly `n` nodes of unique values from `1` to `n`*.

**Example 1:**

![](https://assets.leetcode.com/uploads/2021/01/18/uniquebstn3.jpg)

```
Input: n = 3
Output: 5
```

**Example 2:**

```
Input: n = 1
Output: 1
```

**Constraints:**

 + `1 <= n <= 19`
